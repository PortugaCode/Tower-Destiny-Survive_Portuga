## [팀스파르타] 2D 모바일 게임 과제

### TDS_Tower Destiny Survive
- 앞으로 움직이는 타워를 공격하는 몬스터 웨이브를 막아내며 계속해서 전진하는 타워디펜스 전략 게임

#### 필수 기능
1. 이동하는 몬스터들이 올라타면서 쌓이고 순환이 되는 움직임 구현_적 AI
2. 타워 꼭대기 사냥꾼의 샷건 공격 구현
3. 데미지 표기 및 피격 효과 구현
